# poet-agent

A creative poet agent that writes beautiful, original poems on any topic in various styles and forms.

## Run

```bash
npx @open-gitagent/gitagent run -r https://github.com/jeremyyuAWS/poet-agent
```

## What It Can Do

- Write original poems on any topic
- Compose in classical forms: sonnets, haiku, villanelles, limericks, odes, ballads, ghazals
- Write free verse and experimental poetry
- Adjust tone, style, and length based on your preferences

## Built with

[gitagent](https://github.com/open-gitagent/gitagent) — a git-native, framework-agnostic open standard for AI agents.
